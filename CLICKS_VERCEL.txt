CLIQUE 1: Abra o Projeto
─────────────────────────────────────────────────────────────────────────

Procure por: "pre_validador_sefaz" na lista de projetos
Clique no projeto


CLIQUE 2: Vá para Settings
─────────────────────────────────────────────────────────────────────────

Você será levado à página do projeto

Na parte superior, procure por:
    [ Settings ]

Clique em Settings


CLIQUE 3: Vá para Security
─────────────────────────────────────────────────────────────────────────

Você está em: Project Settings

Na sidebar esquerda (esquerda da página), procure por:
    [ Security ]

Clique em Security


CLIQUE 4: Encontre Deployment Protection
─────────────────────────────────────────────────────────────────────────

Você está em: Settings > Security

Procure por um card/box chamado:
    "Deployment Protection"
    OU
    "Protect Deployments"

Verá algo como:
    ┌────────────────────────────────┐
    │ Deployment Protection          │
    │                                │
    │ Enabled [TOGGLE]               │
    │ (com um botão/switch ativo)    │
    │                                │
    │ Disable (botão)                │
    └────────────────────────────────┘


CLIQUE 5: DESABILITE a Proteção
─────────────────────────────────────────────────────────────────────────

Opção A - Se houver um toggle:
    Clique no toggle para mudar de ON → OFF

Opção B - Se houver um botão "Disable":
    Clique no botão [ Disable ]

Opção C - Se houver checkbox:
    Desmarque a caixa


CLIQUE 6: Confirme (se necessário)
─────────────────────────────────────────────────────────────────────────

Se aparecer uma caixa de diálogo perguntando:
    "Are you sure? This will make..."
    ou
    "Confirm Disable?"

Clique em:
    [ Confirm ]  ou  [ Disable ]  ou  [ Yes ]
